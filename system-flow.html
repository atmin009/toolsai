Zettaword – Simple User Flow

ภาพรวมแบบย่อของการทำงานตามลำดับขั้น จากผู้ใช้จนถึงการ publish บทความ

Start
➜
เปิดหน้าเว็บ Zettaword
➜
หน้า Login
มีบัญชีแล้วหรือยัง?
ถ้า "ยังไม่มี" → สมัครสมาชิก (Register)
⇣
Login เข้าระบบ (ได้ JWT)
⇣
เลือก / สร้าง Website ที่ต้องการทำคอนเทนต์
⇣
ไปหน้า Planner กำหนดเดือน + จำนวนโพสต์/วัน
➜
ระบบเรียก AI สร้าง Topic Plan รายวัน
⇣
อนุมัติหัวข้อแล้วหรือยัง?
ถ้า "ไม่โอเค" → ปรับ / ลบ / Regenerate Topic
⇣
ถ้า "โอเค" → Approve Topic
⇣
จาก Topics Page เลือกหัวข้อ → กด Generate Article
➜
ระบบใช้ AI สร้างร่างบทความ
⇣
เปิด Article Editor แก้ไขเนื้อหา + SEO
➜
บันทึก & ตรวจ checklist คุณภาพ
⇣
พร้อม Publish หรือยัง?
ถ้ายัง → กลับมาแก้เพิ่มภายหลัง
⇣
ถ้าพร้อม → เลือกหมวดหมู่/แท็ก + กด Publish to WordPress
⇣
บทความถูกสร้างบน WordPress แล้ว (มีลิงก์ให้เปิดดู)

Diagram นี้ตั้งใจให้ดู Flow การใช้งานแบบคร่าว ๆ เท่านั้น ไม่ลงรายละเอียดโค้ดหรือ API.
